## 🔍 요청사항
- n개의 이미지가 슬라이드 될 수 있는 영역을 만듭니다.
- 다양한 크기의 이미지를 넣어도 별도 수정 없이 동작이 가능합니다.
    - 이부분 유의하기
- 슬라이드 양 옆에 화살표가 있습니다.
- 화살표를 클릭하면 슬라이드가 작동됩니다.
